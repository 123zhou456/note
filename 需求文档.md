# 日记便签 — 产品需求文档

> 版本 1.2.1 | 2026-06-11  
> Android · 纯离线 · Markdown 笔记

---

## 1. 产品概述

纯离线 Android 笔记应用。编辑 Markdown → 点击预览，不依赖网络，数据存本地。支持自定义背景图与磨砂卡片风格。

### 核心功能

- Markdown 笔记的创建、编辑、预览
- 多级标题折叠展开
- 标签分类与筛选
- 插图（相册选取）与手写
- 图片二次编辑（手写涂鸦）、图片管理（查看/编辑/替换/删除）、图片保存到本地
- 导出 TXT/MD/CSV → 手机文件管理器可见
- 一键同步备份到本地存储
- 深色模式
- 自定义背景图 + 卡片透明度调节

## 2. 界面与交互

### 2.1 笔记列表页

| 区域 | 内容 |
|------|------|
| 顶栏 | 「日记便签」标题、深色模式切换、设置菜单、排序下拉 |
| 标签栏 | 显示所有标签，点击筛选，可多选 AND 过滤 |
| 卡片 | 标题 + 120 字预览 + 标签 + 时间 + 单篇导出按钮 |
| FAB | 右下角 + 号，新建笔记 |
| 空状态 | 无笔记/无匹配时显示提示 |
| 底栏 | 「日记便签」水印 |

**卡片样式**：磨砂玻璃效果（`backdrop-blur`），透明度可在设置中调节（30%–100%），背景图微微模糊。

**卡片左滑删除**：左滑卡片露出红色删除区域；删除按钮仅在左滑时渲染，静止状态下完全不可见。点击卡片进入编辑，再次点击收回删除区。

### 2.2 编辑器

| 区域 | 内容 |
|------|------|
| 顶栏 | 返回、保存状态（已保存/保存中/未保存）、保存按钮 |
| 标题 | 独立输入框 |
| 标签 | Badge 展示，**左滑出现删除按钮**（点击删除）；点击其他区域收起；弹窗添加/创建标签 |
| 工具栏 | H1/H2/H3、加粗、文字颜色（9 色）、插入图片（相册）、手写、撤销 |
| 内容区 | 编辑/预览双模式切换 |

**编辑模式**：全屏 textarea，工具栏可插入 Markdown 语法。

**预览模式**：渲染 Markdown，标题可折叠。点击预览内容进入编辑模式，光标自动定位到点击处附近文字。点击「完成 ✓」切回预览，滚动位置按比例同步。

**自动保存**：编辑时 1.5 秒防抖自动保存。

**颜色语法**：`[c:red]文字[/c]` 渲染为对应颜色文字。

**插图与手写**：

| 功能 | 说明 |
|------|------|
| 插入图片 | 工具栏「图片」按钮 → 从设备相册选取 → 以 base64 存储，编辑器中显示短引用 `![插图](img:uuid)` |
| 手写 | 工具栏「手写」按钮 → 画板手写 → 保存后以 `![手写](hw:uuid)` 插入 |
| 预览渲染 | 预览模式下引用自动解析为实际图片，支持点击放大查看（Lightbox） |

**图片管理与二次编辑**（v1.2.0 新增）：

预览模式下点击已插入的图片，弹出底部操作菜单（ActionSheet），提供以下操作：

| 操作 | 说明 |
|------|------|
| 查看大图 | 全屏 Lightbox 查看图片 |
| 手写编辑 | 进入全屏图片编辑器，在图片上涂鸦手写（画笔/橡皮/撤销），保存后涂鸦合并到原图 |
| 保存到本地 | 将图片保存到设备相册 `Pictures/日记便签/` 目录，首次使用需授权存储权限 |
| 替换图片 | 从相册重新选取图片，替换当前图片（保持原 UUID 不变） |
| 删除图片 | 从笔记内容中移除图片引用及数据 |

**图片编辑器**（`image-editor.tsx`）：全屏覆盖层，双层 Canvas 架构——底层 Canvas 渲染原始图片，上层透明 Canvas 用于手写涂鸦。支持画笔（多色、多粗细）、橡皮擦（仅擦除笔迹不擦除原图）、撤销（逐笔迹撤销）。保存时将涂鸦层合并到底层图片，导出为 JPEG 格式。图片通过 UUID 标识，编辑器通过 UUID 定位并更新对应图片数据。

### 2.3 设置菜单

| 菜单项 | 说明 |
|--------|------|
| 设置背景图 | 从相册选择，设为应用背景 |
| 清除背景图 | 恢复纯色背景（有背景图时显示） |
| 卡片透明度 | 滑块调节卡片不透明度（30%–100%），实时预览，localStorage 持久化 |
| 导入笔记 | 选择文件导入 TXT/MD/CSV |
| 同步到本地 | 一键备份全部笔记为 TXT |
| 存储设置 | 查看用量 + 备份操作 |
| 导出全部 | TXT / MD / CSV 子菜单 |
| 按标签导出 | 选标签 → 选格式（有标签时显示） |
| 标签管理 | 独立管理页 |
| 回收站 | 软删除管理 |

### 2.4 其他页面

| 页面 | 说明 |
|------|------|
| 标签管理 | 创建/重命名/删除标签 |
| 回收站 | 恢复/永久删除/清空 |
| 存储设置 | 显示用量 + 备份全部/独立文件/大文件 |
| 同步到本地 | 一键导 TXT 到 Download/日记便签/ |

## 3. Markdown 输入约定

| 输入 | 渲染结果 |
|------|---------|
| `# 标题` | H1（可折叠） |
| `## 标题` | H2（可折叠） |
| `### 标题` | H3（可折叠） |
| `**粗体**` | 粗体 |
| `*斜体*` | 斜体 |
| `~~删除线~~` | 删除线 |
| `` `行内代码` `` | 行内代码 |
| `[c:red]文字[/c]` | 红色文字 |
| `> 引用` | 引用块 |
| `- 列表` | 无序列表 |
| `1. 列表` | 有序列表 |
| `- [x] 任务` | 任务清单 |
| ` ```代码块``` ` | 代码块 |
| `---` | 分隔线 |
| 回车 | 新段落（自动补 `\n\n`） |

支持 GFM（GitHub Flavored Markdown）扩展语法。

## 4. 导出

### 4.1 命名规则

| 导出范围 | TXT/MD 文件名 |
|---------|--------------|
| 单篇 | `笔记标题.ext` |
| 全部 | `diary.ext` |
| 按标签 | `标签_标签名.ext` |

CSV 始终用相同命名规则。

### 4.2 格式

| 格式 | 元数据 |
|------|--------|
| TXT | `---\ntitle/ tags/ createdAt/ updatedAt\n---` |
| MD | YAML front matter |
| CSV | 表头 id,title,content,created_at,updated_at,tags |

多篇用 `=====` 分隔。

### 4.3 存储位置

手机：`Download/日记便签/`，通过文件管理器可见。

### 4.4 单篇导出

卡片左滑区域或卡片内导出按钮触发，弹出格式选择浮层（TXT/MD/CSV）。

---

## 5. 导入

支持导入 TXT / MD / CSV，自动解析标签，自动创建不存在的标签，无标签笔记导入后自动标记「未分类」。

---

## 6. 撤销

工具栏 `↩`：2 秒停顿自动快照 + 手动操作前快照，最多 50 步。

---

## 7. 目录 (TOC)

右下角浮动按钮，展开后显示标题树，点击跳转到对应位置。编辑模式下定位光标到标题行，预览模式下滚动视图。无标题时按钮不显示。

---

## 8. 深色模式

通过顶栏月亮/太阳图标切换。全局支持，包括磨砂卡片、编辑器、预览、弹窗等所有界面。

---

## 9. 首次使用

首次打开应用（无任何笔记时）自动创建示例笔记「Markdown 使用指南」和「入门」标签，引导用户了解基本功能。示例内容可在 `src/lib/api-handler.ts` 的 `DEMO_NOTE_CONTENT` 中修改。

---

## 10. 存储

| 项目 | 说明 |
|------|------|
| 数据库 | IndexedDB `diary_notes_db` |
| 位置 | WebView 沙盒（卸载清除） |
| 表 | notes, tags, noteTags, settings |
| 导出 | Download/日记便签/ |
| 图片存储 | 笔记 `images` 字段（JSON，base64 data URL） |
| 图片保存 | `Pictures/日记便签/`（用户手动保存的图片，需存储权限） |
| 用户偏好 | localStorage（卡片透明度等） |
